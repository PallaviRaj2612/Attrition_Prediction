Employee Attrition Prediction Hackathon
Project Overview
This project focuses on predicting employee attrition using machine learning techniques. 
The goal is to identify employees at risk of leaving the organization based on various personal, professional, and performance-related factors.

Dataset
The dataset contains employee information including demographic details, job satisfaction, work-life balance, and performance metrics. Key features include:
Age
Monthly Income
Job Satisfaction
OverTime
Involvement
Other HR-related factors
The target variable is Attrition

Problem Statement
Employee attrition is costly for organizations in terms of recruitment, training, and lost productivity. This project aims to:
Analyze factors affecting attrition.
Build a predictive model to identify employees likely to leave.
Provide actionable insights for HR management.

EDA (Exploratory Data Analysis)
Checked for missing values and handled them.
Visualized feature importance and correlations.
Identified top factors contributing to attrition, e.g., OverTime, Age, Job Satisfaction, Monthly Income.

Modeling
Multiple machine learning models were tried including:
Random Forest
XGBoost
Logistic Regression

Model evaluation metrics used:
Accuracy
Precision
Recall
F1 Score

Feature Importance

Top features impacting attrition prediction:
OverTime
Age
Involvement
Monthly Income
Job Satisfaction

Results
Best model achieved Accuracy = 91% and high precision, indicating reliable predictions for identifying employees at risk.
Feature analysis provided actionable insights for HR strategies.
